# EXAMPLES DISTRICT ROLE

## PURPOSE

The `/examples` district is intentionally different from the main portal shell.

It exists as:
- a live workshop
- a client preview space
- a works-in-progress gallery
- a visual sandbox
- a proof-of-work district
- a rapid experimentation zone

## IMPORTANT

Examples should NOT become a fully rigid synchronized ecosystem shell.

The value comes from:
- visible experimentation
- evolving layouts
- active iteration
- project previews
- unusual showcase energy
- client-facing flexibility

## KEEP

- personality
- experimentation
- color freedom
- fast-moving layouts
- showcase atmosphere
- project discovery focus

## STILL REQUIRE

- mobile sanity
- footer consistency
- ecosystem links
- navigation clarity
- reduced duplicate clutter
- readable hierarchy
- working routes
